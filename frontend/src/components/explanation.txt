************************************
FOLDER: components
************************************
DESCRIPTION:
This folder contains reusable, generic components 
that are not tied to any specific feature. They are 
modular and can be used across multiple parts of the app.

EXAMPLES:
- Button/
  A button that can be reused throughout the app.
  Example: Button.jsx, Button.module.css
- Header/
  The app's main header with a navigation menu.
  Example: Header.jsx, Header.module.css
- Footer/
  The footer that displays copyright or app info.
  Example: Footer.jsx, Footer.module.css
