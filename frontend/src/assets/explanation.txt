************************************
FOLDER: assets
************************************
DESCRIPTION:
This folder stores static files used throughout the app, 
such as images, icons, and global styles. These files 
are not dynamic and are shared across components and 
features.

EXAMPLES:
- images/
  For visual elements like backgrounds or logos.
  Example: background.jpg, logo.png
- icons/
  For icons used in buttons or navigation.
  Example: weather-icon.svg, menu-icon.svg
- styles/
  For global or shared CSS files.
  Example: variables.css, global.css
************************************
