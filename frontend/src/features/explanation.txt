************************************
FOLDER: features
************************************
DESCRIPTION:
This folder organizes major functionalities or views 
into separate modules. Each feature contains its own 
components, styles, and utilities.

EXAMPLES:
- MapView/
  Displays an interactive map.
  Includes: MapView.jsx, MapMarker.jsx, mapHelper.js
- WeatherDetails/
  Displays detailed weather information.
  Includes: WeatherDetails.jsx, weatherDataFormatter.js
************************************
